简介与安装

1.简介：
Git是一种的分布式版本控制工具,开发者是Linux操作系统作者Linus Torvalds
初衷是为了更好的管理Linux内核，而现在却早已被广泛应用全球各种大中小型项目中

2.安装
Ubuntu系统:打开shell，输入sudo apt-get install git-core
Window系统：访问网址http://mysgit.github.io/,点击中央的Downloads,下载安装包，然后一直Next
