1.设置java代码运行，快捷键

Eclipse，点击顶部Window —-> Preferences ———->General —–>Keys
———->右界面可滑动部分，找到R开头，选中Run Java Application
———>光标移动到下面的Building,按下F6【设置F6为快捷键】——–>OK
