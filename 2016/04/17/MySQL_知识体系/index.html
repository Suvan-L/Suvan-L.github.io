4. 约束

  • AUTO_INCREMENT 【主键自增，默认起始值为1，增量1】
    • PRIMARY KEY 【主键约束，表中唯一，NOT NULL】
  • UNIQUE KEY 【唯一约束，记录唯一】
    • DEFAULT 【默认约束，自动赋予默认值】
  • FOREIGN KEY 【外键约束，一对一 or 一对多】
